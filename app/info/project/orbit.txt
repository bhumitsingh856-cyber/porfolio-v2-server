post: Orbit – Full Stack Social Media Application

Orbit is a production-style social media platform built using the MERN stack.
The focus is on scalability, real-world features, and clean UI/UX.

✨ Core Features:
• 🔐 User authentication & authorization (JWT)
• 📝 Create posts with images and videos
• ⏳ Stories feature
• ➕ Follow / Unfollow system
• 👤 User profiles with dynamic data
• 💬 Real-time one-to-one chat (Socket.io)
• 🤖 AI-powered chat assistant with text and image generation 
• ⚡ Lazy loading & skeleton loaders
• 📱 Fully responsive design (mobile & desktop)

🛠 Technical Highlights:
• REST APIs with Express & MongoDB
• ☁️ Media handling via Cloudinary
• 🔄 Real-time communication using Socket.io
• 🔍 Debounced search
• 🍪 Secure cookie-based authentication
• 🧩 Modular & reusable UI components

🧑‍💻 Tech Stack:
Frontend: React, Tailwind CSS, Framer Motion, Swiper js 
Backend: Node.js, Express, MongoDB 
Other Tools: Cloudinary, Socket.io, Gemini API, Stable Diffusion XL
